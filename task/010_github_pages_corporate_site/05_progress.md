       112 +  ## 🚀 Автоматизация и CI/CD
       113 +  
       114 +  ### Repository Dispatch Events
       115 +  Федерация автоматически обновляется при изменениях в любом продуктовом репозитории:
       116 +  
       117 +  ```yaml
       118 +  on:
       119 +    repository_dispatch:
       120 +      types: [
       121 +        corporate-site-updated,      # info-tech
       122 +        quiz-docs-updated,           # quiz
       123 +        hugo-docs-updated,           # hugo-templates
       124 +        web-terminal-docs-updated,   # web-terminal
       125 +        cli-docs-updated,           # info-tech-cli
       126 +        complete-federation-update   # manual trigger
       127 +      ]
       128 +  ```
