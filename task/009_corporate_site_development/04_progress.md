## Техническая инфраструктура

### DNS конфигурация
```
info-tech.io              → A → [IP VPS]
*.info-tech.io            → A → [IP VPS] (wildcard)
quiz.info-tech.io         → CNAME → info-tech.io
hugo.info-tech.io         → CNAME → info-tech.io
terminal.info-tech.io     → CNAME → info-tech.io
```

### Apache2 конфигурация
- Поддержка множественных доменов
- SSL терминация для всех поддоменов
- Роутинг через mod_rewrite
- Security headers настроены

### Файловая структура
```
/var/www/info-tech.io/
├── index.html              # Корпоративный сайт
├── quiz/                   # Quiz Engine docs
├── hugo/                   # Hugo Templates docs
├── terminal/               # Web Terminal docs
└── assets/                 # Общие ресурсы
```
