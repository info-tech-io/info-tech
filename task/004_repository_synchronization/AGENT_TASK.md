# Задача: Синхронизация репозиториев проекта info-tech-io

**Цель:** Обновить все локальные репозитории проекта, приведя их в соответствие с удаленными версиями на GitHub.

**Основные шаги:**
1.  **Анализ:** Составить список всех локальных git-репозиториев проекта.
2.  **Проверка:** Проверить `git status` каждого репозитория на наличие незакоммиченных изменений.
3.  **Синхронизация:** Выполнить `git fetch` и `git pull` для каждого репозитория, чтобы загрузить последние изменения.
4.  **Верификация:** Убедиться, что все репозитории успешно обновлены и не имеют конфликтов.
