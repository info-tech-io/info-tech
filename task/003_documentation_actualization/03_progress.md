┌─────────────────────────────────────────────────────────┐
│                 INFOTECHA.RU PLATFORM                   │
│  ┌─────────────────┐  ┌─────────────────┐               │
│  │   Central Hub   │  │  Module System  │               │
│  │  (infotecha)    │  │   (mod_*)       │               │
│  └─────────────────┘  └─────────────────┘               │
└─────────────────────────────────────────────────────────┘

### 2. Актуализированная доменная структура
```
infotecha.ru                    # Главный хаб
├── linux-base.infotecha.ru      # Основы Linux
├── linux-advanced.infotecha.ru  # Продвинутый Linux

```

- Hugo Static Site Generator

- Let's Encrypt SSL

### 4. Фактическая структура репозиториев
