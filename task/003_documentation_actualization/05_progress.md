## Критерии принятия решений

### ✅ Оставить документ, если:
- Содержит уникальную актуальную информацию
- Не дублирует другие документы
- Соответствует текущему состоянию проекта
- Имеет практическую ценность

### 🔄 Обновить документ, если:
- Содержит ценную информацию с неточностями
- Использует устаревшую терминологию
- Требует обновления URL или технических деталей

### 🗑️ Удалить документ, если:
- Полностью дублирует другие документы
- Содержит безнадежно устаревшую информацию
- Не имеет практической ценности для проекта
